# Employee Management System

A Full Stack Employee Management Application built using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.

---

# Features

✅ Admin Login & Logout  
✅ Add Employee  
✅ View Employees  
✅ Update Employee  
✅ Delete Employee  
✅ Search Employees  
✅ Dashboard Analytics Cards  
✅ Dark Mode  
✅ Form Validation  
✅ Toast Notifications  
✅ Responsive UI  
✅ MySQL Database Integration

# How to Run the Project

## Step 1
Clone the repository

```bash
git clone https://github.com/harish-kakumani08/employee-management-system.git
```

---

## Step 2
Open project in IntelliJ IDEA or VS Code

---

## Step 3
Configure MySQL Database in:

```text
application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/employeedb
spring.datasource.username=root
spring.datasource.password=yourpassword
```

---

## Step 4
Run the Spring Boot Application

---

## Step 5
Open Browser

```text
http://localhost:8082
```
